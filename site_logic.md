# Willnicht - Логика сайта и User Journey

## 🎯 Цель продукта
SaaS-платформа для автоматической оценки товаров с помощью AI. Пользователь загружает фото товаров и получает:
- Описание предмета
- Рыночную цену
- Рекомендуемую цену для маркетплейса

---

## 👤 Путь пользователя (User Journey)

### Этап 1: Первое знакомство (Landing)
```
[Пользователь заходит на сайт]
         ↓
[Видит Hero-секцию с описанием сервиса]
         ↓
[Понимает ценность: "Оцени товар за секунды"]
         ↓
[CTA: "Начать оценку" / "Загрузить фото"]
```

### Этап 2: Загрузка товара
```
[Нажимает "Загрузить фото"]
         ↓
[Открывается модальное окно / страница загрузки]
         ↓
[Drag & Drop или выбор файлов (до 10 фото)]
         ↓
[Превью загруженных фото с возможностью удаления]
         ↓
[Кнопка "Получить оценку"]
```

### Этап 3: Обработка
```
[Фото отправляются на webhook]
         ↓
[Показывается индикатор загрузки с анимацией]
         ↓
[AI анализирует изображения]
         ↓
[Получение ответа от сервера]
```

### Этап 4: Результат
```
[Отображается карточка товара с результатом]
         ↓
┌─────────────────────────────────────┐
│  [Фото товара]                      │
│  ─────────────────────────────────  │
│  📝 Описание: "Винтажные часы..."   │
│  💰 Рыночная цена: $150-200         │
│  🏷️ Рекомендуемая цена: $179        │
│  ─────────────────────────────────  │
│  [Добавить ещё] [Сохранить]         │
└─────────────────────────────────────┘
```

### Этап 5: Управление коллекцией
```
[Пользователь видит все свои оценённые товары]
         ↓
[Может фильтровать/сортировать по цене]
         ↓
[Может экспортировать данные]
         ↓
[Может удалить или редактировать товар]
```

## 📱 Структура страниц

### 1. Главная страница (/)
- **Header**: Логотип, навигация, CTA-кнопка
- **Hero**: Заголовок + подзаголовок + кнопка действия
- **Features**: 3 преимущества сервиса (иконки + текст)
- **How it works**: 3 шага работы с сервисом
- **Pricing** (опционально): Тарифные планы
- **Footer**: Контакты, ссылки, копирайт

### 2. Страница загрузки (/upload)
- Зона drag & drop для фото
- Превью загруженных изображений
- Кнопка отправки на анализ
- Индикатор прогресса

### 3. Страница результатов (/results)
- Сетка карточек с оценёнными товарами
- Каждая карточка: фото + описание + цены
- Фильтры и сортировка
- Кнопка "Добавить ещё"

### 4. Страница деталей товара (/item/:id)
- Полноразмерное фото (галерея если несколько)
- Детальное описание от AI
- Рыночная цена с диапазоном
- Рекомендуемая цена для продажи
- Кнопки действий: редактировать, удалить, экспорт

## 🔄 Состояния интерфейса

### Загрузка фото
| Состояние | Описание | UI элемент |
|-----------|----------|------------|
| Idle | Ожидание загрузки | Drag & drop зона |
| Uploading | Файл загружается | Progress bar |
| Processing | AI анализирует | Spinner + текст |
| Success | Результат готов | Карточка товара |
| Error | Ошибка | Сообщение + retry |

### Список товаров
| Состояние | Описание | UI элемент |
|-----------|----------|------------|
| Empty | Нет товаров | Placeholder + CTA |
| Loading | Загрузка списка | Skeleton cards |
| Loaded | Товары отображены | Grid карточек |
| Filtering | Применяется фильтр | Анимация |

## 🎨 Дизайн-система (на основе solt.ws)

### Цветовая палитра
```css
--color-background: #FFFFFF;      /* Основной фон */
--color-text: #000000;            /* Основной текст */
--color-accent: #FF3B30;          /* Акцент (кнопки, цены) */
--color-secondary: #F5F5F5;       /* Вторичный фон */
--color-border: #E5E5E5;          /* Границы */
--color-success: #34C759;         /* Успех */
--color-muted: #8E8E93;           /* Приглушённый текст */
```

### Типографика
- **Заголовки**: Grotesk/Inter Bold
- **Основной текст**: Inter Regular
- **Акценты**: Inter Medium

### Компоненты
- Карточки с тенью и скруглёнными углами
- Кнопки с hover-эффектами
- Плавные анимации (ease-out)
- Модальные окна для загрузки

## 📊 Технический поток данных

```
┌──────────────┐     ┌─────────────┐     ┌──────────────┐
│   Frontend   │────▶│   Webhook   │────▶│   AI Model   │
│  (Загрузка)  │     │  (Backend)  │     │  (Анализ)    │
└──────────────┘     └─────────────┘     └──────────────┘
       ▲                    │                    │
       │                    ▼                    │
       │            ┌─────────────┐              │
       └────────────│   Response  │◀─────────────┘
                    │  (JSON)     │
                    └─────────────┘
```

### Формат ответа от webhook:
```json
{
  "success": true,
  "items": [
    {
      "id": "uuid",
      "image_url": "...",
      "description": "Описание товара от AI",
      "market_price": {
        "min": 150,
        "max": 200,
        "currency": "USD"
      },
      "recommended_price": 179,
      "confidence": 0.85,
      "category": "Часы"
    }
  ]
}
```
